一个真正大幅提升浏览器体验的软路由网关。

极大提高了dns性能，优化nginx，支持http2，tls1.3等特性。

![de_GWD](https://i.loli.net/2019/10/13/lc8eJs1dt2hIRNT.png)

## Server：

```
apt install -y wget
bash <(wget --no-check-certificate -qO- https://raw.githubusercontent.com/gwdburst/de_GWD/master/de_GWD_Server/server)
```
